# 外部接口测试模式

> 适用场景：任何涉及第三方接口调用、外部系统集成的需求
>
> 进化记录：
> - v1.0：初始版本，5个测试模式

---

## 必测场景清单

### 1. 接口调用

| 测试场景 | 测试要点 |
|---------|---------|
| 正常调用 | 接口正常返回数据，数据解析正确 |
| 接口超时 | 超时后重试机制验证、重试次数限制 |
| 接口失败 | 失败后告警触发、状态处理、用户提示 |
| 返回数据异常 | 数据格式异常、字段缺失处理 |
| 接口限流 | 第三方限流时的处理逻辑 |

### 2. 数据安全

| 测试场景 | 测试要点 |
|---------|---------|
| 数据加密存储 | 敏感字段加密存储验证 |
| 传输加密 | HTTPS 传输验证、证书校验 |
| 日志脱敏 | 日志中敏感信息脱敏验证 |
| 数据过期处理 | 过期后自动删除/状态翻转 |
| 数据访问权限 | 数据访问权限控制验证 |

### 3. 有效期管理

| 测试场景 | 测试要点 |
|---------|---------|
| 有效期边界 | 有效期最后一天（23:59:59）仍可用 |
| 过期翻转 | 过期后（00:00:00）状态自动翻转为失效 |
| 配置生效 | 有效期天数配置生效验证 |
| 配置热加载 | 配置修改后热加载生效 |
| 跨时区处理 | 不同时区下的有效期计算 |

### 4. 授权/认证

| 测试场景 | 测试要点 |
|---------|---------|
| 协议展示 | 协议内容正确、签署流程完整 |
| 授权过期处理 | 授权过期后重新授权流程 |
| 重新授权流程 | 重新授权后接口调用是否正常发起 |
| 多机构授权 | 多个第三方授权管理、授权状态独立 |
| 授权范围限制 | 授权范围是否覆盖本次调用类型 |
| 授权取消 | 用户取消授权后的处理逻辑 |

### 5. 告警验证

| 测试场景 | 测试要点 |
|---------|---------|
| 告警触发条件 | 接口失败、超时自动触发告警 |
| 告警内容验证 | 告警内容包含：关键信息、失败原因、时间 |
| 告警渠道验证 | 告警渠道验证（IM、短信、邮件等） |
| 告警去重 | 同一失败不重复告警（去重机制） |
| 告警恢复 | 问题恢复后告警状态更新 |

## 测试数据准备建议

### 有效数据示例

| 数据类型 | 示例 | 说明 |
|---------|------|------|
| 请求参数 | 符合接口规范的参数 | 需准备真实格式数据 |
| 返回数据 | 正常返回数据 | 模拟第三方返回 |
| 授权凭证 | 有效授权凭证 | 验证调用正常 |

### 异常数据示例

| 数据类型 | 示例 | 说明 |
|---------|------|------|
| 返回空 | `{}` 或 `null` | 验证空数据处理 |
| 字段缺失 | 缺少关键字段 | 验证字段缺失处理 |
| 格式异常 | JSON格式错误 | 验证解析异常处理 |

## 与其他模式的关系

| 关联模式 | 关联场景 |
|---------|---------|
| 状态流转测试模式 | 接口状态 → 业务状态 → 过期状态 |
| 通用测试模式 | 告警验证、配置验证、权限验证 |
