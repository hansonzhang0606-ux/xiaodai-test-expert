## 🔍 使用说明

### AI 搜索历史文档

1. **业务领域匹配**：当前需求属于某领域 → 搜索该领域下的文档
2. **功能相似匹配**：当前需求涉及某功能 → 搜索包含该功能的历史文档
3. **关键词匹配**：从当前需求名称提取关键词 → 搜索关键词匹配的文档

### 文档命名规范

| 文档类型 | 命名格式 |
|---------|---------|
| 需求文档 | `{编号}_{名称}_{日期}.md` |
| 评审报告 | `{编号}_{名称}_{日期}_评审报告.md` |
| 技术方案 | `TR_{编号}_{名称}_{日期}.md` |
| 测试点 | `{编号}_{名称}_{日期}_测试点.json` |
| 测试用例 | `{编号}_{名称}_{日期}_测试用例.json` |
